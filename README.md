# 📱 Klasifikasi Teks: Deteksi Spam SMS dengan Machine Learning

Repositori ini memuat kode sumber (Jupyter Notebook) dan dataset yang digunakan untuk menyelesaikan tugas Natural Language Processing (NLP). Fokus utama dari repositori ini adalah membangun *pipeline* klasifikasi teks untuk membedakan pesan SMS biasa (*Ham*) dan pesan sampah (*Spam*).

## 🚀 Cakupan Proyek

Proyek ini mendemonstrasikan tahapan NLP secara lengkap, yang terdiri dari:
1. **Pengumpulan Data:** Analisis awal pada *SMS Spam Collection dataset*.
2. **Pra-Pemrosesan Teks (Text Pre-processing):** Implementasi *lowercasing*, pembersihan tanda baca, tokenisasi, penghapusan *stopwords*, dan *stemming*.
3. **Rekayasa Fitur (Feature Engineering):** Ekstraksi fitur menggunakan **TF-IDF** dan *word embedding* dengan **Word2Vec**.
4. **Exploratory Data Analysis (EDA):** Visualisasi distribusi panjang teks, frekuensi kata teratas (*bar plot*), dan *Word Cloud*.
5. **Pemodelan & Klasifikasi:** Pelatihan dan evaluasi model *machine learning* (seperti Naive Bayes, SVM, dan Logistic Regression) beserta komparasi metrik (*Accuracy, Precision, Recall, F1-Score*).
6. **Kajian Teori:** Pembahasan akademik mengenai perbedaan *Text Classification* dan *Text Clustering*, evaluasi *Elbow Method*, dan *Silhouette Score*.

## 🛠️ Teknologi & *Library* yang Digunakan
* **Bahasa:** Python 3.x
* **Data Manipulation:** Pandas, NumPy
* **NLP Tools:** NLTK, Gensim
* **Machine Learning:** Scikit-Learn
* **Visualization:** Matplotlib, Seaborn, WordCloud
