Repositori ini memuat kode sumber (Jupyter Notebook) dan dataset yang digunakan untuk membangun *pipeline* Natural Language Processing (NLP). Fokus utama proyek ini adalah melakukan klasifikasi teks untuk membedakan pesan SMS biasa (*Ham*) dan pesan sampah (*Spam*) menggunakan pendekatan *Supervised Learning*.

## 🚀 Cakupan Proyek

Proyek ini mendemonstrasikan tahapan NLP secara menyeluruh, meliputi:
1. **Identifikasi Masalah:** Mengangkat studi kasus dari literatur mengenai deteksi SMS Spam.
2. **Pengumpulan Data:** Eksplorasi pada *SMS Spam Collection dataset*.
3. **Pra-Pemrosesan Teks (Text Pre-processing):** Implementasi *lowercasing*, pembersihan tanda baca/angka/URL, tokenisasi, penghapusan *stopwords*, dan *stemming*.
4. **Rekayasa Fitur (Feature Engineering):** Ekstraksi fitur teks menggunakan **TF-IDF** dan pembuatan *word embedding* dengan **Word2Vec**.
5. **Exploratory Data Analysis (EDA):** Visualisasi distribusi panjang teks, frekuensi kata teratas (*bar plot*), dan *Word Cloud*.
6. **Pemodelan & Klasifikasi:** Pelatihan, komparasi, dan evaluasi model *Machine Learning* (Naive Bayes, SVM, dan Logistic Regression) berdasarkan metrik *Accuracy, Precision, Recall*, dan *F1-Score*.
7. **Kajian Teori:** Pembahasan komprehensif mengenai perbedaan *Text Classification* dan *Text Clustering*, evaluasi *Elbow Method*, dan *Silhouette Score*.
